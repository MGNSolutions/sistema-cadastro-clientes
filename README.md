## ▶️ Como executar

1. Clone este repositório:

```bash
git clone https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git
```

2. Acesse a pasta do projeto:

```bash
cd Cadastro-Clientes
```

3. Instale a biblioteca necessária:

```bash
pip install openpyxl
```

4. Execute o projeto:

```bash
python main.py
```
